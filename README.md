## Test Task Sendway

### Инструкция по установке:

В cmd
```
git clone https://github.com/Crysknight/test-task-sendway.git
cd test-task-sendway
npm i
npm start
```

Обратите внимание, что, чтобы посмотреть, как работает нотификация, необходимо пропустить ответный звонок от контакта (подождать 5 секунд).
